Greedy Algorithm

Picks the cell that covers the most uncovered neighbors at each step. Fast but may not always give the global optimum.

Time: O(N4) Space: O(N2)

Dynamic Programming

Splits grid into regions using flood-fill, then solves each region as an independent sub-problem for better placement.

Time: O(N4) Space: O(N2)
